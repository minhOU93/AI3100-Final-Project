# AI3100-Final-Project
Using [Nerfstudio](https://github.com/nerfstudio-project/nerfstudio) to reconstruct the Senior Design Room via a NeRF and exporting its mesh to be used in Panda3D

## How to Use

### Run from source

1. Clone the repo: `git clone https://github.com/minhOU93/AI3100-Final-Project.git`

2. Pip install the requirements: `pip install -r requirements.txt` (you can also just do `pip install panda3d`)

3. Run the program: `python main.py`

### Download Release

Alternatively, you can download the binaries from the releases.
